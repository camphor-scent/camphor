Getting Started
--
- Install Nodejs
- fork the reposiotry to your local system
- Navigate to project folder using Command Line
- run `npm install`
- run 'npm install -g bower'
- run 'bower install'
- finally run `node server.js` to bring up the  demo app and access `https://localhost`
